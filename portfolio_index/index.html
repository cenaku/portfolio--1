    画像処理実習 — 不思議の国のアリス展

    AI生成の門番・壁・猫を一枚にまとめた合成。素材ごとに違う色味を馴染ませる「色合わせ」が最大の難所で、寄せ集めでなくダークで没入感のあるアリス展の一場面として成立させた。ロゴはIllustratorで担当。 ツール：VS Code / Illustrator / Photoshop / ChatGPT / LEONARDO.AI

  • 03 — HTML / CSS · UX Design · AI

    総合実習 — 京都市情報館

    3C・STP分析をもとにコンセプトを策定し、e-Estoniaを参考にしたタスクファースト設計で京都市公式サイトをリデザイン。「市民が用事を済ませて、早く閉じられる」行政ポータルを目指し、ターゲット分析から設計・コーディングまで一貫して制作した。 ツール：VS Code / HTML・CSS・JavaScript

    • Photoshop / Ai

      ドリルレコードロゴデザイン

      板橋区のバー「ドリルレコード」オープン時に、オーナーから依頼を受けて制作した実案件のロゴ。90年代グラフィックを参照しつつ、看板・Tシャツでも形が崩れないシンプルさで仕上げた。展開イメージは AI 生成のモックで確認。

    • Retouch / Color Grading

      画像加工実習 — リタッチ・色校正

      人物写真のリタッチと、風景写真の色校正2点。素材の雰囲気を壊さない範囲で意図した印象に寄せる、最小限の調整に徹した作業セット。
